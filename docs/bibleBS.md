# 📘 КНИГА КАБАЛКА (KABALOK BIBLE)
## Офіційна документація та посібник користувача для казкового симулятора (Версія 3.0 - Фінальний Реліз)

> **Цільове призначення документа:** Цей документ є офіційним виробничим стандартом та посібником для проєкту **КАБАЛОК (KABALOK)**. Він містить повний опис ігрового процесу, етно-дизайну персонажів, фізичних колізій, процедурної генерації рівнів та преміальної мобільної адаптації гри. Цей документ дозволяє будь-якому розробнику ознайомитися з архітектурою та механіками гри для її підтримки та запуску.

---

## ── РОЗДІЛ 1: ЗАГАЛЬНЕ БАЧЕННЯ ТА КОНЦЕПТ ──

### 1.1. Короткий концепт (High Concept)
**КАБАЛОК (KABALOK)** — це атмосферна, яскрава та супер-захоплива браузерна аркадна гра-платформер, натхненна безсмертною українською народною казкою про Колобка. 

Назва гри є даниною поваги класичному герою **Колобку**, який весело котиться вперед, долаючи будь-які перешкоди на своєму шляху.

Гравець керує круглим, рум'яним та веселим героєм КАБАЛКОМ, який котиться по мальовничих пагорбах, перестрибує небезпечні урвища, збирає золоті зірочки-монети та намагається втекти від хижих лісових мешканців — Кібер-Лисиці та Кібер-Вовка.

Гра пропонує нескінченний аркадний режим із процедурною генерацією рівнів на основі кімнат-патернів, де складність зростає з кожною пройденою локацією.

### 1.2. Основні стовпи дизайну (Design Pillars)
1.  **Казкова атмосфера та етно-кіберпанк стиль:** Візуальний стиль гри поєднує яскраві елементи традиційної української казки (сонце, зелені пагорби, квіти) із сучасними неоновими акцентами та геометричним мінімалізмом.
2.  **Соковите та інтуїтивне керування:** Рух героя використовує м'яку інерцію, пружну деформацію при стрибках та нахил корпусу при швидкому бігу, що дарує гравцеві максимальне задоволення від кожного руху.
3.  **Адаптивний нелінійний левелдизайн:** Рівні розширюються не лише по горизонталі, але й по вертикалі (шахти, підйоми, глибокі колодязі), пропонуючи різноманітні шляхи для збору монет та ключів.
4.  **Ергономічна мобільна сумісність:** Повноекранний геймплей (100% Edge-to-Edge) з великими тач-зонами під великі пальці обох рук, тактильною вібрацією та оптимізованим інтерфейсом для будь-яких екранів.

## ── РОЗДІЛ 2: ХУДОЖНЯ НАРАТИВНІСТЬ ТА ПЕРСОНАЖІ ──

### 2.1. Світ Гри
Дія гри розгортається у дивовижному чарівному лісі. Мальовничі пагорби вкриті яскравими квітами, у небі світить лагідне сонце, а час відлічує старовинний лісовий годинник. Проте за простою оболонкою ховається загадкова геометрична краса: ліани схожі на шини зв'язку, а глибокі колодязі зберігають цінні таємниці лісу.

### 2.2. Дійові особи
*   **🟡 КАБАЛОК (Колобок):** Головний герой. Має вигляд класичного Колобка — жовто-оранжева сфера з теплим 3D-градієнтом, великими анімованими очима (з чорними зіницями та білими відблисками), рум'яними рожевими щічками та веселою червоною посмішкою. Він котиться вперед, виконуючи високі пружні стрибки.
*   **🦊 Кібер-Лисиця (aluPatrol):** Хитрий математичний патруль лісу. Виглядає як яскрава помаранчева лисичка з трикутними вушками, білими пухнастими щічками, світящимися салатовими очима та білим символом `+` на лобі. Швидко патрулює платформи та заважає герою котитися.
*   **🐺 Кібер-Вовк (GCPatrol):** Величний та небезпечний охоронець глибоких рівнів лісу. Має форму великого темно-сірого вовчого блоку з гострими вухами, чорною мордою та червоними світлодіодними очима. Зверху вниз випускає фіолетовий промінь світла, якого герою слід уникати.

## ── РОЗДІЛ 3: ГЕЙМПЛЕЙНІ МЕХАНІКИ ТА ФІЗИЧНИЙ РУШІЙ ──

### 3.1. Нескінченна аркада (Casual Loop)
Основний ігровий цикл побудований на проходженні рівнів:
*   Гравець має знайти на рівні ключ (або кілька ключів) та дістатися до вихідних воріт.
*   На платформах розкидані золоті обертові зірочки-монети, які збільшують фінальний рахунок.
*   Наприкінці кожного рівня показується красива картка результатів з рейтингом від 1 до 3 зірок, який розраховується на основі часу проходження та кількості зібраних монет.

### 3.2. Поведінка при Смерті (Reset Behavior)
При загибелі КАБАЛОКА (падіння на шипи, дотик до ворогів або обнулення HP):
*   Герой миттєво відроджується на початку поточного рівня.
*   Його здоров'я (HP) відновлюється до максимальних 3-х одиниць.
*   Раніше зібрані ключі та монети на поточному рівні повертаються на свої місця, даючи можливість пройти рівень заново.

### 3.3. Керування та Преміальна Адаптивність
*   **Реєстрація імені:** На початку гри впроваджено обов'язкове поле введення імені оператора в неоново-бірюзовому скляному стилі. Якщо поле порожнє, гра блокує запуск режимів, відтворює системний звук помилки та запускає горизонтальну вібрацію-трясіння контейнера введення.
*   **Сенсорні кнопки на мобільних пристроях:**
    *   *Ліва сторона:* Кнопки руху вліво та вправо.
    *   *Права сторона (Верх):* Кнопка стрибка.
    *   *Права сторона (Низ):* Кнопка дії (ACTION / удар молотом для захисту від ворогів).
    *   *Верхній центр:* Кнопка встановлення бар'єра (BLOCK).
*   **Панель дій:** Круглі напівпрозорі комірки розташовані в нижній центральній частині екрана. Це звільняє ігрове поле і покращує огляд платформ на телефонах.
*   **Універсальне прокручування:** Усі екрани меню, налаштувань та досягнень автоматично підтримують вертикальне прокручування на екранах з низькою висотою (менше 550px), що робить інтерфейс максимально ергономічним.
*   **Тактильний відгук (Vibration API):** При колізіях, отриманні шкоди або стрибках мобільний телефон видає приємну мікро-вібрацію (`20ms-100ms`).

### 3.4. Фізичні константи
Рух гравця використовує м'яку інерцію та силу тертя. Параметри налаштовуються в коді `js/game.js`:

```javascript
// Конфігураційні фізичні константи симуляції
const PHYSICS = {
  GRAVITY: 2200,             // Сила тяжіння (пікселів/сек²)
  JUMP_FORCE: 650,           // Початкова вертикальна швидкість стрибка
  MAX_SPEED: 450,            // Максимальна швидкість бігу по горизонталі
  ACCELERATION: 1200,        // Прискорення при русі
  GROUND_FRICTION: 0.92,     // Сила тертя на землі (множник швидкості)
  AIR_FRICTION: 0.99,        // Сила опору повітря
  NORMAL_JUMP_HEIGHT: 96     // Висота стандартного стрибка (пікселів)
};
```

## ── РОЗДІЛ 6: ТЕХНІЧНА АРХІТЕКТУРА ──

Всі зміни та механіки інтегровані в наступні файли:
1.  **`index.html`:** Фронтенд гри. Повноекранний Canvas, тач-зони, HP-бар, панель 3-х слотів дій, діалоговий консольний термінал.
2.  **`style.css`:** Стилі інтерфейсу, неонові градієнти, адаптивні мобільні медіа-запити для прокручування екранів.
3.  **`js/game.js`:** Ядро. Ігровий цикл (60 FPS), фізика, керування станами, обробка колізій, генерація рівнів, підрахунок зірок.
4.  **`js/renderer.js`:** Етно-кіберпанк візуалізатор Canvas 2D. Рендеринг КАБАЛОКА (Колобка з очима/посмішкою), Кібер-Лисиці, Кібер-Вовчика, палітр та ефектів.
5.  **`js/levels.js`:** Генератор кімнат-патернів та функції процедурного створення рівнів.
